Switch Statement

A switch statement functions like many nested if statements.

switch(case int) //a case value is passed into the switch statement
{
case 1:
case 2:
//do something
break; //in this case, case 1 and case 2 will both trigger the same discrete case statement.
case 3:
//do something else
break;
default: //this statement is called when the value of 'case' is not one of the above specified cases.
//this statement should deal with exceptions, etc.
break;
